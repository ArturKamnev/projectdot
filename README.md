# projectdot

Этот Python-скрипт позволяет управлять курсором мыши жестами рук через веб-камеру.
Используются:
- MediaPipe для отслеживания руки
- OpenCV для захвата видео
- PyAutoGUI для эмуляции мыши

## Требования
- Python 3.x
- opencv-python
- mediapipe
- pyautogui

## Запуск
1. pip install -r requirements.txt
2. python projectdot.py
